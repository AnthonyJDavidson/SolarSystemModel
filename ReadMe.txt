** info on how/why it looks how it does **

The ratio of the planet's radius' is correct
The ratio of the orbits are also correct

The two however are not correct together as this would create a fairly blank scene
and a lot of scrolling to find the next planet

The sun is not correct at all as it's sheer size would have thrown off the entire simulation

Clicking on a planet will give some information on the planet
If the information doesn't show, zoom in a little
